AnyBody Tutorials!¶

Welcome to the AnyBody Tutorials. This is a good place to start learning how to do musculoskeletal modeling in the AnyBody Modeling System.

You can find the available tutorials in the sidebar. The tutorials are ordered in a suitable sequence for new users who are unfamiliar with AnyBody, but this sequence may not be optimal for you depending on your background and interests.

It is recommended to start with the three Getting Started tutorials.

Note

If you are a new user, please spend 7 minutes to watch the introductory video above, before beginning the tutorials.

Contributing

We highly appreciate any contributions to Tutorials!

If you find typos, missing links or anything else wrong, don’t hesitate to fix it. It is as easy as forking the repository on GitHub , making your changes, and issuing a pull request.

More details are in the how to contribute section.
